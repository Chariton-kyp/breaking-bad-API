Welcome To Breaking Bad API Project

So what are we going to see?

The whole project is going to be implemented with the use of:
Bootstrap 5,simple JavaScript for functionallity and Ajax for API calls

In this page, You will see how the whole project is going to be implemented.

The Home link in the the Navigation Bar shows the current page and it is the introduction.

The Characters link drives you to the second page, where all the characters of Breaking Bad will be shown.

In this page you will see the images of all the characters, their name and their nickname.

When you press at a character, a bootstrap modal will appear which will have at it's left side, the character's attributes.

At it's right side, all the quotes of that specific character will appear in a list.
